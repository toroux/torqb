# How to Install the New Interface Style for RTX TV

This guide explains how to install the new interface style for the RTX TV product.

Files Needed:

* scripts.js
* styles.css
* ui.html

Installation Steps:

1. Go to the rtx_tv/html folder.

2. Paste the new files (scripts.js, styles.css, ui.html) from the "New Interface Style" folder into the rtx_tv/html folder.

3. Replace the existing files if prompted.


Notes:

* All files must be placed directly into the rtx_tv/html folder.
* Make sure the file names are correct.
